Azure based ETL and Pipeline project

This project implements an end-to-end ETL (Extract, Transform, Load) pipeline using Azure Data Factory, Blob Storage, and Azure SQL Database. The pipeline ingests data from CSV files stored in Blob Storage, applies optional data transformations (e.g., filtering, column mapping), and loads the cleansed data into a structured SQL database for reporting and analytics. It leverages linked services, parameterized datasets, and scheduling triggers to automate the data integration process in a scalable and maintainable fashion.
